同入参格式，形如：

```json
{
  "id": "<短链接对象的 ID>",
  "url": "<目标 URL>",
  "key": "<短链接码>",
  "tag": "<标签>",
  "redirectType": "<跳转方式>",
  "expiredAt": "<过期时间>",
  "public": false,

  "$reuse": false,
  "$full": "<完整的短链接 URL>"
}
```

需要注意的是：

- 短链接码 `key` 使用时，需拼接 URL 前缀；
- 额外字段 `$reuse` 表示是否复用了现有短链接；
- 额外字段 `$full` 表示带有前缀的短链接。
